# Security Policy

Security documentation for this project is maintained in:

```text
docs/SECURITY.md
docs/SECURITY-CHECKLIST.md
```

## Reporting a Vulnerability

Do not open a public GitHub issue for a suspected vulnerability.

Please use GitHub Security Advisories for vulnerability reporting.

This project is provided as open source software under the Apache License 2.0, without warranties. Security reports are reviewed on a best-effort basis and no response time guarantees are provided.

Please do not disclose suspected vulnerabilities publicly.

Please include:

affected version
deployment context
reproduction steps
impact assessment
whether user authentication data, credentials, TOTP seeds, authentication sequences, WebAuthn credentials, or API tokens may have been exposed


## Supported Versions

Only the latest published release is expected to receive security fixes unless the release notes state otherwise.
