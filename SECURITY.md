# Security Policy

Security documentation for this project is maintained in:

```text
docs/SECURITY.md
docs/SECURITY-CHECKLIST.md
```

## Reporting a Vulnerability

Do not open a public GitHub issue for a suspected vulnerability.

Report security issues to the project maintainers through the official
Shinshu University contact channel for this plugin. The public repository
should replace this paragraph with the final security contact address before
the first public release.

Please include:

- affected version
- deployment context
- reproduction steps
- impact assessment
- whether credentials, TOTP seeds, GraphicalMatrix sequences, or API tokens may
  have been exposed

## Supported Versions

Only the latest published release is expected to receive security fixes unless
the release notes state otherwise.
