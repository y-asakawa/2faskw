# Contributing

Thank you for considering a contribution to GraphicalMatrix MFA for Shibboleth IdP.

## License

By submitting a contribution, you agree that your contribution is licensed
under the Apache License, Version 2.0, the same license as this project.

## Security Issues

Do not report vulnerabilities in public issues or pull requests. Follow
`SECURITY.md` instead.

## Pull Requests

- Keep changes focused.
- Include tests or verification steps when possible.
- Do not include real IdP configuration secrets, private keys, certificates,
  TOTP seeds, database dumps, exported CSV files, or production logs.
- Do not commit generated release ZIP files unless the release process
  explicitly requires it.
